Hello!

My name is Fabrício Matté (a.k.a. UltCombo). I am an ECMAScript enthusiast, full stack Web developer and open source lover.

I maintain the Harmonic project, curate JS Rocks and work as a back-end Web developer at Nodo. I also contribute to Stack Overflow:

~/

  • Creator and maintainer of slush-es20xx, a fully optimized ECMAScript.next development and deployment workflow.
  • Maintainer of Harmonic, a static site generator developed using the latest ECMAScript features, which runs on the Node.js and io.js platforms. Built on top of slush-es20xx.
  • Curator, writer and copywriter of JS Rocks, a collaborative website about the latest JavaScript features and tools. Built with Harmonic.
  • Creator and maintainer of gulp-buster, a blazing fast and fully configurable cache buster hashes generator for gulp.
  • Creator and maintainer of poly-checked, a miraculous :checked pseudo-class polyfill for IE8 and below.
  • Creator and maintainer of UltVoice, a library to implement voice commands using the Speech Recognition API.
  • Creator and maintainer of blink-tag, a polyfill to enable the <blink> tag in modern browsers.
  • Author of WTHPL, the true freedom license.
  • Creator of the (now deprecated) traceuroso, an ECMAScript.next workflow for developing modular packages with no build steps.
  • Creator of the (now deprecated) Sublime-SemicolonifyJS, a Sublime Text 2/3 plugin which automatically semicolonifies your JS files upon save.
  • Creator of the (now deprecated) UltButtons, a project to improve the jQuery UI Checkbox and Radio Buttons’ usability.
